About us

KiNG

KiNG (Kinemage, Next Generation) is an interactive system for three-dimensional vector graphics. It supports a set of graphics primitives that make it suitable for many types of graphs, plots, and other illustrations; although its first use was to display macromolecular structures for biophysical research. KiNG builds on Mage, JavaMage, and the "kinemage" (kinetic image) concept to deliver a full-featured Java application with a user-friendly interface and integrated editing features. It also operates as a Java applet to promote easy access to kinemages from a web browser.

KiNG is written in Java and should run on all platforms with a Java Virtual Machine (aka Java Runtime Environment) of version 1.3.1 or higher. Yet, in the table below are entries for specific platform applications - Linux, Windows, Mac OSX. All Mac OSX computers include a Java Virtual Machine (JVM), so either the Mac OSX App bundle (a “double-clickable” application) or the “any platform” jars (OS9 or OSX) will be the appropriate download for the Macintosh platform. A Java VM is not standardly installed on the Linux or Windows OS. So, the Linux and Window platforms downloads are bundled with a JVM, which will only be used as part of KiNG. These two packages are targeted for computers which do not already have Java installed. If your Linux or Windows computer does have a 1.3.1 or later JVM, then you'll want to download the “any platform” jars instead.

To download: click the “KiNG” name for the appropriate table-row. Depending upon your browser set-up you may need to "option-click" or “right-click” and “save-as”
